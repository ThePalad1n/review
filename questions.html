Section 2

Comprehension Check

You will be employer-ready if you can answer the following questions:

  1. What is a CDN?
  2. What is a CSS framework?
  3. What are three benefits of using a CSS framework?

Learning Objectives

You will be employer-competitive if you are able to:

  1. Explain the importance of CSS resets and cross-browser compatibility
  2. Explain and implement content delivery network links in HTML files
  3. Implement a grid layout in HTML to leverage the benefits of a CSS framework
  4. Integrate Bootstrap components in HTML files to quickly build user interfaces
  5. Add custom styling to a CSS framework
  6. Deploy to GitHub pages
